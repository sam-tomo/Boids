# Boids
「動物の群れ行動のシミュレーションに関する研究」

こちらは本研究で構築したシミュレーターのソースコードおよび実行結果です。

## シミュレーターの使用方法（論文より一部引用）

### 動作環境
本研究で作成した群れ行動シミュレーターを実行するために必要な動作環境を示す。
OSはWindows10/11を想定する。実行環境としてJuliaを用いる。
本シミュレーターはJulia標準ライブラリであるRandom、LinearAlgebra、Statisticsに加え、
描画のためPlots、表示制御のためPrintfを使用する。また、描画にはPlots.jlのバックエンドとしてGRを用いる。

### ダウンロード方法
GitHubを通じて群れ行動シミュレーターのソースコードをダウンロードすることが可能である。
本シミュレーターは以下の3つのソースコードで構成される。

  params.jl：シミュレーションのパラメータ(例：エージェントの総数、領域サイズ、最大速度など）を設定する。

  Boids.jl：シミュレーションのメイン処理および可視化・出力を行う。

  obstacles.jl：障害物機能の定義および関連処理をまとめている。

これらのファイルをダウンロードすることで本シミュレーターを扱うことが可能である。

### 初期設定
本シミュレーターはJulia上で動作する。開発環境としては VSCode 上での実行を想定するが，
Julia が動作する環境であれば同様に実行可能である。
初回実行時には、描画に用いるパッケージPlotsを事前に導入する必要がある。
なお，Random，LinearAlgebra，Statistics，Printf は Julia の標準ライブラリであり，追加導入は不要である．

### 実行方法
VSCode 上で本シミュレーターを実行する手順を示す．

  1. params.jl，Boids.jl，obstacles.jl の3つのファイルを同一フォルダに配置し，当該フォルダを作業ディレクトリとして実行する．

  2. Boids.jl を開く．

  3. VSCode 右上の実行メニューから 「Julia: Execute Active File in REPL」 を選択する．

この手順により、シミュレーターが実行され、シミュレーション結果が描画される。

### 出力内容
本シミュレーターを実行すると，シミュレーション結果はファイルとして保存される．
実行完了後，VSCode の Julia REPL 上に Finished: result files saved. と表示され，
以下の 5 つの結果ファイルが作業ディレクトリ（Boids.jl を配置したフォルダ）に保存される．

  Boids.gif：メトリック型およびトポロジカル型の群れ挙動を示すアニメーション．

  order.svg：秩序度の時系列変化を示すグラフ．

  nn.svg：平均最近傍距離の時系列変化を示すグラフ．

  cluster.svg：最大クラスタサイズの時系列変化を示すグラフ．

  speed_std.svg：速度の標準偏差の時系列変化を示すグラフ．

### 課題・注意点
・出力関数の影響により、シミュレーション領域を拡大すると、各エージェントの表示も連動して
　拡大されてしまいます。そのため、個体数は1~200個を推奨としています。
・障害物の大きさや進行方向等の設定はobstacles2.jlではなく、Boids2.jlでの呼び出しで設定可能です。
　色の変更や定義の追加などはobstacles2.jlから変更・拡張可能です。
・乱数生成シード値の変更はパラメータ設定では変更できません。Boids2.jlから
　Random.Seed!(...)の値を変更してください。
